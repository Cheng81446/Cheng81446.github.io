# 中国茶文化博客

这是一个使用 **Jekyll + GitHub Pages** 搭建的个人博客网站，分享中国芳茶、名人茶事与心芝术。

## 🌿 网站内容
- **茶叶介绍**：绿茶、红茶、乌龙茶、白茶等分类与品鉴
- **名人茶事**：陆羽、苏轼、弘一法师等与茶的故事
- **心芝术**：茶与冥想、养生、心灵修养的结合
- **个人博客**：记录我的学习心得与生活感悟

## 🚀 如何访问
网站地址：[https://Cheng81446.github.io](https://Cheng81446.github.io)

## 🛠️ 技术说明
- 使用 [Jekyll](https://jekyllrb.com/) 生成静态网站
- 部署在 [GitHub Pages](https://pages.github.com/)
- 支持 Markdown 撰写文章，存放在 `_posts` 文件夹
